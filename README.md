1. **Intermediate C#** - OOP and Structured Programming (09-22)

1. **Advanced C#** (23-46)
2. **Design Principles** (01-09)
3. **LINQ** (01-20)
4. **Unit Testing** (01-05)

**Best Practices:**
- Review **Code Takeaways** throughout your journey
- Apply **Design Principles** in your projects
- Practice **Unit Testing** from intermediate level onwards
